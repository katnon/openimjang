mapprime Example

3D 객체 생성 도구
  • 도형 그리기 타입별(점,선,면) 도형그리기 예제
  • Point(점) Entity생성 포인트를 활용한 다양한 Entity생성
  • Line(선) Entity생성 입력받은 좌표값으로 라인을 생성
  • Polygon(면) Entity생성 입력받은 좌표값으로 폴리곤을 생성
  • Entity객체 삭제 입력한 도형 객체를 삭제
  • 피라미드 생성 입력한 지점에 피라미드 도형을 생성
  • 모델 Transformer 모델객체의 위치, 크기, 회전값을 수정
  • SHP파일 업로드 shp파일 건물세우기
환경 효과
  • 날씨(비) 효과 비 효과(Rain Effect)를 화면에 추가
  • 날씨(눈) 효과 눈 효과(Snow Effect)를 화면에 추가
  • 안개 안개 효과(Fog Effect)를 화면에 추가
분석 가시화
  • 경사도 분석 입력받은 영역의 경사도를 분석
  • 바람길 분석 입력한 범위에 바람길을 분석
  • 토공량 분석 영역의 굴착시각화와 토공량을 분석
  • 스카이 라인 현재 지도화면의 스카이라인
  • 종횡단면 분석 종단면과 횡단면 각각의 높이를 분석
  • 시곡선 분석 시점에서 종점까지 시곡선을 기준으로 지형,건물,시야 방해 여부를 분석
  • 음영 분석 햇빛이 가려지는지의 여부를 분석
  • 일조량 분석 선택한 영역에 대해 설정한 시간 동안의 일조량을 분석
  • 가시권 분석 선택한 시점에서 종점을 기준으로 가시권 지역과 비가시권 지역을 분석
